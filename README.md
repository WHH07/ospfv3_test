# 配置
使用三个frr容器t0, t1, t2。拓扑如下：
```
     eth02              eth30
 2001:3000::11      2001:3000::13
+----+                         +----+
| t0 | ======================= | t3 |
+-++-+                         +----+
  ||     eth01
  || 2001:0000::11
  ||
  ||     eth10
  || 2001:1000::10
+-++-+
| t1 |
+-++-+
  ||     eth11
  || 2001:1000::11
  ||
  ||     eth20
  || 2001:2000::10
+----+                         +----+
| t2 | ======================= | t4 |
+-++-+                         +----+
     eth21              eth40
 2001:4000::10      2001:4000::14


```

# 启动！
```bash
./test.sh run
```

# 测试
```bash
ip netns exec t3 ping -6 2001:4000::14
```
发现从t0可以ping通t2的eth40口。

# 清理
```bash
./test.sh rm
```
